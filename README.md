# vue-ts-demo
vue3.0+ant-design-vue2.2.8搭建的脚手架，接口按照RESTful API规范请求。
表格只需json配置配置配置就完事，内置表格内各种组件修改，基本满足大部分需求。

## Project setup
```
npm install
```
